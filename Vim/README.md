# Vim cheatsheet

## Folds

| Key | Action |
|-----|--------|
| `zi` | Toggle all folds on/off |
| `za` | Toggle fold under cursor |
| `zR` | Open all folds |
| `zM` | Close all folds |
| `zj` / `zk` | Jump to next / previous fold |

> Run `vimtutor` to refresh the basics. Create a Vim deck on Anki.
